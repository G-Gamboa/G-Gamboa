## Proyectos destacados

**[GGStore](https://github.com/G-Gamboa/GGStore)** · Next.js, TypeScript, Prisma, PostgreSQL, Cloudinary, Docker
Tienda en línea en producción. Catálogo con filtros, carrito integrado a WhatsApp, panel administrativo con autenticación, dashboard de métricas e importación masiva por CSV. [Demo](https://www.ggstore.lat/)

**[SistemaDicri](https://github.com/G-Gamboa/SistemaDicri)** · React, Express, SQL Server, Docker Compose, JWT
Plataforma fullstack para gestión de expedientes e indicios criminalísticos. Autenticación por roles, flujo de aprobación, historial de estados y stored procedures.

**[BotFinanzasAPI](https://github.com/G-Gamboa/BotFinanzasAPI)** · Python, FastAPI, SQLAlchemy, Pydantic, OAuth2
API REST para finanzas personales. Gestión de transacciones, categorías y presupuestos con validación rigurosa y arquitectura modular.

**[BotFinanzasFront](https://github.com/G-Gamboa/BotFinanzasFront)** · React, TypeScript, Vite, Tailwind CSS, GitHub Actions
Cliente del ecosistema BotFinanzas. Interfaz conversacional y tableros de indicadores con tipado estricto y pipeline de CI configurado.

**[BotFinanzas](https://github.com/G-Gamboa/BotFinanzas)** · Python, python-telegram-bot, Google Sheets API
Bot conversacional para registro y consulta de finanzas. Arquitectura modular con handlers, validaciones, jobs programados y persistencia en Sheets.

**[Detecci-nCrisis](https://github.com/G-Gamboa/Detecci-nCrisis)** · Python, BERT, HuggingFace, Flask
Sistema de apoyo a líneas de ayuda psicológica que estima riesgo de crisis emocional en llamadas mediante NLP.
